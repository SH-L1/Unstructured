# 도로 파손 민원 처리 매뉴얼

## 문서 유형
민원 처리 매뉴얼

## 적용 민원 유형
도로 파손, 포트홀, 도로 균열, 인도 파손, 차량 통행 위험, 보행 위험

## 담당 부서
도로관리과

## 처리 절차
1. 도로 파손 민원을 접수한다.
2. 민원인이 신고한 위치를 확인한다.
3. 차량 통행 또는 보행 안전에 위험이 있는지 검토한다.
4. 긴급 보수가 필요한 경우 우선 조치 대상으로 검토한다.
5. 현장 확인 후 보수 일정 또는 처리 방향을 안내한다.

## 민원 답변 활용 기준
도로 파손 민원이 접수된 경우, 공문 초안에는 불편과 안전 우려에 대한 사과, 현장 확인 예정, 보수 필요성 검토, 처리 방향 안내 내용을 포함한다.

## 공문 초안 작성 시 주의사항
보수 완료를 단정하지 않는다.
현장 확인 전에는 정확한 공사 일정이나 보수 완료 일자를 확정하지 않는다.
"현장 확인 후 보수 필요성을 검토하겠습니다"라고 작성한다.
