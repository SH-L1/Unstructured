# 생활폐기물 및 불법 투기 민원 처리 매뉴얼

## 문서 유형
민원 처리 매뉴얼

## 적용 민원 유형
불법 투기, 생활폐기물 무단 배출, 폐가구 방치, 악취 민원

## 담당 부서
자원순환과

## 처리 절차
1. 민원 내용을 접수한다.
2. 민원 발생 위치를 확인한다.
3. 현장 확인이 필요한 경우 담당자가 위치를 확인한다.
4. 생활폐기물, 폐가구, 불법 투기 여부를 검토한다.
5. 필요 시 수거 조치 또는 단속 절차를 검토한다.
6. 민원인에게 처리 예정 사항 또는 확인 결과를 안내한다.

## 민원 답변 활용 기준
불법 투기 또는 생활폐기물 관련 민원이 접수된 경우, 공문 초안에는 민원 접수 사실, 불편에 대한 사과, 현장 확인 예정, 관련 절차에 따른 조치 검토 내용을 포함한다.

## 공문 초안 작성 시 주의사항
처리 완료를 단정하지 않는다.
과태료 부과 여부를 단정하지 않는다.
"현장 확인 후 관련 절차에 따라 검토하겠습니다"와 같이 작성한다.
